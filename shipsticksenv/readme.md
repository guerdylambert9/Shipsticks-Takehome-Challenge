# Shipsticks-Takehome-Challenge

Automated test suite for the ShipSticks Senior QA Engineer take-home challenge.

## Prerequisites

- **Python 3.9+** (3.10+ recommended for latest pytest-playwright)
- Git

## Setup

1. **Clone the repo** (if needed) and go into the test project:
   ```bash
   cd shipsticksenv
   ```

2. **Create and activate a virtual environment:**
   ```bash
   python3 -m venv .
   source bin/activate   # On Windows: .\Scripts\activate
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Install Playwright browsers** (required for UI tests):
   ```bash
   playwright install
   ```

5. **Optional — set base URL**  
   Tests use `https://app.staging.shipsticks.com` by default. To override, create a `.env` file in `shipsticksenv/`:
   ```bash
   BASE_URL=https://app.staging.shipsticks.com
   ```

## How to run the tests

From the **`shipsticksenv`** directory with the virtual environment activated:

**Happy-path suite (recommended):**
```bash
pytest tests/ -m happy_path --headed
```

**All tests:**
```bash
pytest tests/ --headed
```

**Headless (no browser window):**
```bash
pytest tests/ -m happy_path
```

**With HTML report** (generated by default; open `report.html` after the run):
```bash
pytest tests/ -m happy_path --headed
# Report: shipsticksenv/report.html
```

## Test markers

- `happy_path` — main booking flow (landing → address → item → date → service)
